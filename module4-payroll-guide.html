Formula Kalkulasi Gaji
KOMPONEN
FORMULA / NILAI
Gaji Pokok
employee.salary_base
Tunjangan Transport
+Rp 300.000
Tunjangan Makan
+Rp 450.000
Lembur (jika ada)
+(jam × gaji/jam × 1.5)
Koreksi Absen Tidak Sah
-(hari × gaji/hari)
= Gaji Kotor
SUM pokok + tunjangan
BPJS Kesehatan
-1% dari gaji pokok
BPJS TK / JHT
-2% dari gaji pokok
PPH 21
-5% dari (kotor - PTKP 4.5jt)
Potongan Terlambat
-Rp 25.000 × jumlah terlambat
= Take Home Pay
Kotor − Total Potongan

04
UI Components — PayrollPage.jsx
🧾 SlipModal (Bottom Sheet)
Slide-up modal fullscreen pada mobile. Header gradient purple/biru dengan nama, NIP, jabatan. Take Home Pay besar. Breakdown lengkap: rekap kehadiran, tabel komponen gaji dengan ikon +/−, tabel potongan merah, dan total akhir hijau.
📊 AdminTab — Kelola Gaji
Month navigator. Summary cards (total gaji + status count). Tombol Generate (kalkulasi otomatis) + Bayar Semua (bulk pay). List karyawan dengan gaji, status badge, tombol lihat slip + mark paid.
📄 MyPayrollTab — Slip Saya
Year selector + total take-home tahunan. Bar chart 12 bulan dengan warna: hijau (paid), amber (processed), abu (kosong). List slip per bulan, klik buka SlipModal detail lengkap.
📈 Dashboard Stats Live
Keempat stat card di dashboard sekarang menampilkan data live: absensi hari ini, sisa cuti tahunan, total gaji bulan ini (HR/admin), dan jam kerja hari ini.

Test Commands
BASHGenerate payroll bulan ini (semua karyawan)
curl -X POST http://localhost:5000/api/payroll/generate \
  -H "Authorization: Bearer HR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"month":"2024-01"}'
BASHGenerate untuk karyawan tertentu saja
curl -X POST http://localhost:5000/api/payroll/generate \
  -H "Authorization: Bearer HR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"month":"2024-01","user_ids":[3,4]}'
BASHList payroll bulan ini (HR)
curl "http://localhost:5000/api/payroll?month=2024-01" \
  -H "Authorization: Bearer HR_TOKEN"

curl -X POST http://localhost:5000/api/payroll/bulk-pay \

  -d '{"month":"2024-01"}'
